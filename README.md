# CRUD API with Node.js & MongoDB

## Features
- Create, Read, Update, Delete tasks
- MongoDB with Mongoose
- Error handling middleware
- Logging with Morgan

## Setup
1. Clone repo
2. npm install
3. Create .env file from .env.example
4. npm run dev

## API Endpoints
GET /api/tasks
GET /api/tasks/:id
POST /api/tasks
PUT /api/tasks/:id
DELETE /api/tasks/:id
